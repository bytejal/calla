# 👗 Calla

Calla is a digital wardrobe management app built with Flutter. It helps users organize their clothes and, in the future, will provide AI-powered outfit recommendations.

## Features

- Splash Screen
- Onboarding
- Login & Sign Up
- Home Screen

## Tech Stack

- Flutter
- Dart

## Getting Started

```bash
git clone https://github.com/bytejal/calla.git
cd calla/apps/mobile
flutter pub get
flutter run
```
